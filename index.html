Html On One Page!

On this web-page, we will be learning about useful html elements


We will be learning about:

  • <p> elements
  • <h1-h6> elements
  • <img> elements
  • <iframe> elements

  1. <p> elements

    <p> elements is generally used for the main content of a web page. It represents the paragraphs. <p> elements contain normal text.

  2. <h1-h6> elements

    <h1-h6> elements are headers. <h1> contains the largest text and <h6> contains the smallest text. All <h> have bold text.

    This is an <h1> tag.

  3. <img> elements

    <img> elements are images. <img> elements have an attribute called src. The src is where the image is being pulled from. You can pull an image locally from your computer or from the internent.

    <img src="bob_the_builder.jpg"/> is pulling an image from your file locally.

    <img src="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRojOunWn-_b6oMG-gD36ZRkXiRfAMcccvSpO1Br7iXliUVHLJG"/> is pulling an image from the internet.

    Bob_The_Builder pulled from the internet

  4. <iframe> elements

    <iframe> elements allows you to pull content from other website. However, not all website allow you to pull their content. One good use for <iframe> elements is to pull videos on your web-page. For example, I can pull the new Halo Infinite trailer using

    <iframe width="560" height="315" src="https://www.youtube.com/embed/Fmdb-KmlzD8" frameborder="0" allow="autoplay; encrypted-media" allowfullscreen></iframe>
